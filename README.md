## M_STORE

### Introduction

> M_Store is a male e-commerce site that gives customers an opportunity to shop online for male products. 
> The app is built in Python .

### Installation  
* git clone https://github.com/M0Hassan/M_Store.git
* cd M_Store/app
* Run app.py 
 

### Technology Stack Used

#### Frontend
* HTML5
* CSS3 
* JavaScript
* Bootstarp

#### Backend
* Python - Flask
* MySQL
